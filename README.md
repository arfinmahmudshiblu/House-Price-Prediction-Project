# House-Price-Prediction-Project

# How to run?

```bash 
git clonehttps://github.com/arfinmahmudshiblu/House-Price-Prediction-Project.git
```

```bash
conda create -n webapp python=3.12 -y
```

```bash
conda activate webapp
```

```bash
pip install -r requirements.txt
```
